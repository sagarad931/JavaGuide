Break Statement

The break statement is used to exit a loop or switch statement. It stops the loop immediately, and the control of the program moves outside loop. 

          
            for(int i=1; i<=100; i++){
                if(i==5){
                    break;
                }
                System.out.println(i);
            }
         
     
It will print 1 to 4.

Continue Statement

The continue statement skips the current iteration of a loop . It will bypass the statement of the loop. 

          
            for(int i=1; i<=100; i++){
                if(i==5){
                    continue;
                }
                System.out.println(i);
            }
         
     
It will print 1 to 100 but not 5.
