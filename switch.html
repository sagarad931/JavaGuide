Switch Case In Java

Switch case is used to execute the code block based on the condition.

Switch Case Syntax

             
                // Switch case is used to execute the code block based on the condition.
                switch(expression) {
                    case value1:
                        statement1;
                        break;
                    case value2:
                        statement2;
                        break;
                    case value3:
                        statement3;
                        break;
                    default:
                        statement4;
                }

Example of Switch Case

             
                int noOfDays = 2;
                switch (noOfDays){
                    case 1:
                        System.out.println("The day is sunday.");
                        break;
                    case 2:
                        System.out.println("The day is monday.");
                        break;
                    case 3:
                        System.out.println("The day is tue.");
                        break;
                    default:
                        System.out.println("Invalid option given");
