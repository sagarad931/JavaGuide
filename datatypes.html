Data Types in Java

Data types in Java are used to store information in different ways. They are categorized into two categories:
1. Primitive Data Types
2. Reference Data Types

Primitive Data Types

Primitive data types are the data types that can be stored without the use of any other data types. They are:

  • Integers: They are used to store whole numbers. They have a range from -2,147,483,648 to 2,147,483,647.
  • Doubles: They are used to store decimal numbers. They have a range from -1,797,693,231,104 to 1,797,693,231,103.
  • Longs: They are used to store large numbers. They have a range from -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807.
  • Booleans: They are used to store true or false values. They have only two values, true and false.
  • Chars: They are used to store single characters. They have a range from 0 to 65535.
  • Bytes: They are used to store byte data. They have a range from -128 to 127.

Reference Data Types

Reference data types are the data types that can be stored by the use of other data types. They are:

  • References: They are used to store references to other objects. They are used to store variables.
  • Arrays: They are used to store multiple values in one variable. They are used to store collections of data.
