Looping

In Programming looping are used to repeat the block of the code. For e.g if you want to print your name 100 times then you can use loop. It will saves your time.

Types of Loop

  • For Loop
  • While Loop
  • Do While Loop

For Loop

Syntax

          
            for(initialization; condition; increment/decrement){
            statement1;
            statement2;
            statement3;
            statement...n;
            }

Example to Print 1 to 100 Using For Loop

          
            for(int i=1; i<=100; i++){
                System.out.println(i);
                }

Example to Print 100 to 1 Using For Loop

          
            for(int i=100; i>=1; i--){
                System.out.println(i);
                }

While Loop

Syntax

          
            while(condition){
                statement1;
                statement2;
                statement3;
                statement...n;
               }

Example to Print 1 to 100 Using While Loop

          
            int i =1;
            while(i<=100){
                System.out.println(i);
            i++;
            }

Example to Print 100 to 1 Using While Loop

          
            int i =10;
            while(i>=1){
                System.out.println(i);
            i--;
            }

Do While Loop

Syntax

          
            do{
                statement1;
                statement2;
                statement3;
                statement...n;
            }while(condition);

Example to Print 1 to 100 Using Do While Loop

          
            int i =1;
            do{
                System.out.println(i);
            i++;
            }while(i<=100);

Example to Print 100 to 1 Using Do While Loop

          
            int i =10;
            do{
                System.out.println(i);
            i--;
            }while(i>=1);
