String in Java

A string is a sequence of characters. It is used to store textual data.

Types of Strings

  • String Literal: A string literal is a sequence of characters enclosed in double quotes. It is used to store textual data.
  • String Object: A string object is an object of the java.lang.String class. It is used to store textual data.
          
            // String Literal
            String str = "Hello World";

            // String Object
            String str1 = new String("Hello World");

Methods of String

  • charAt(): It is used to get a character at specified index.
  • concat(): It is used to concatenate strings.
  • equals(): It is used to check if two strings are equal or not.
  • length(): It is used to get the length of a string.
  • replace(): It is used to replace a string with another string.
  • substring(): It is used to get a substring from a string.

String Methods Example In Java

           
                // String Literal
                String str = "Hello World";
    
                // String Object
                String str1 = new String("Hello World");
    
                // charAt()
                System.out.println(str.charAt(0));
    
                // concat()
                System.out.println(str.concat(" ").concat("Java"));
    
                // equals()
                System.out.println(str.equals(str1));
    
                // length()
                System.out.println(str.length());
    
                // replace()
                System.out.println(str.replace("World", "Java"));
    
                // substring()
                System.out.println(str.substring(0, 5));
