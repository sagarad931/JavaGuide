Variables

Variables are container used to store value in program. There are different types of variables where we can store different types of values.

          
            E.g 
            int a = 10;
            String name = "Bimal";
            // here a and name are variables.

Types of Variables:

  • String: For storing text values. E.g. "Ram" [Must be in double quotes]
  • int: For storing integer value. [Decimal is not included] e.g. 10, -10, 8555
  • float: For storing floating point value. [Decimal is included] e.g. 10.0, -10.2, 85.698
  • char: For storing single character. E.g. 'A', 'a', 'z' [Must be in single quotes]
  • boolean : For storing true or false value.

How to Declare Variables: [Syntax]

type variableName = value;

E.g: String address = "USA";
System.out.println(address);

E.g: int num1 = 10;
System.out.println(num1);

E.g: float price = 1008.52;
System.out.println(price);

E.g: char gender = 'M';
System.out.println(gender);

E.g: boolean isMarried = false;
System.out.println(isMarried);

Always use descriptive variable name. Dont use a, b, c because this will make your code more complex.

Variables [Rules in Java]

  • Declare type first. e.g int i =1;
  • Variable name should not be start with number.
  • Variable name should not be keywords. [already defined words]
  • Variable name should not contain space.
  • Variables names are case sensitive.
