Comments

Comments are not executed by the computer. They are only used to explain the code.

Types of Comments

  • Single Line Comment: For commenting single line of code. E.g. // This is a single line comment.
  • Multi Line Comment: For commenting multiple lines of code. E.g. /* This is a multi line comment. */
    •           
            // This is a comment
            System.out.println("Hello World");

            /* This is a multi line comment.
            You can also write here.
            */
