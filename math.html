Math in Java

Math is used to perform mathematical operations.

Math Syntax

Math is used to perform mathematical operations.

         
            => Math.abs(x)
            => Math.ceil(x)
            => Math.floor(x)
            => Math.round(x)
            => Math.max(x, y)
            => Math.min(x, y)
            => Math.pow(x, y)
            => Math.sqrt(x)
            => Math.random()

Math Examples

Method Description Example
Math.abs(x) Returns the absolute value of the given number. Math.abs(-5) = 5
Math.ceil(x) Returns the smallest integer greater than or equal to the given number. Math.ceil(4.4) = 5
Math.floor(x) Returns the largest integer less than or equal to the given number. Math.floor(4.4) = 4
Math.round(x) Returns the nearest integer to the given number. Math.round(4.4) = 4
Math.max(x, y) Returns the largest of the two given numbers. Math.max(4, 5) = 5
Math.min(x, y) Returns the smallest of the two given numbers. Math.min(4, 5) = 4
Math.pow(x, y) Returns the x to the power of y. Math.pow(4, 2) = 16
Math.sqrt(x) Returns the square root of the given number. Math.sqrt(16) = 4
Math.random() Returns a random number between 0 and 1. Math.random() = 0.9

Code Example Java Math

         
            public class MathExample {

                public static void main(String[] args) {
                   System.out.println("Absolute value of -5 is " + Math.abs(-5));
                    System.out.println("Ceiling of 4.4 is " + Math.ceil(4.4));
                    System.out.println("Floor of 4.4 is " + Math.floor(4.4));
                    System.out.println("Round of 4.4 is " + Math.round(4.4));
                    System.out.println("Max of 4 and 5 is " + Math.max(4, 5));
                    System.out.println("Min of 4 and 5 is " + Math.min(4, 5));
                    System.out.println("4 to the power of 2 is " + Math.pow(4, 2));
                    System.out.println("Square root of 16 is " + Math.sqrt(16));
                    System.out.println("Random number is " + Math.random());
                }
            }
