If Condition in Java

If condition is used to execute the code block if the condition is true.

Types of If Condition

  • If-Else Condition: If condition is used to execute the code block if the condition is true, else condition is used to execute the code block if the condition is false.
  • If-Else-If Condition: If condition is used to execute the code block if the condition is true, else if condition is used to execute the code block if the condition is false and else if condition is used to execute the code block if the condition is false.
       
         // If condition is used to execute the code block if the condition is true.
         if(condition) {
             System.out.println("Hello World");
         }

Example of If-Else Condition

       
        int age = 12;

        if(age <=17){
